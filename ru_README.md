# 📱 Mini-Terminal v0.1-prealpha

Простой учебный терминал, написанный на C++.

---

## ✨ Что нового

- Первый push на GitHub
- Поддержка Linux систем

---

## ⚙️ Особенности

- Структурированный и читаемый код  
- Event System в стиле игровых движков 😏  
- Полностью открытая архитектура без скрытых систем  
- Лёгкая расширяемость  
- Простая и понятная логика исполнения команд  

---

## 📁 Структура проекта

```txt
source/
    core/
        events.cpp     <- ключевая система событий
        variables.cpp  <- работа с буфером и состоянием
        compare.cpp    <- функции сравнения

    include/
        colors.hpp     <- цвета текста (ANSI)
        compare.hpp    <- функции сравнения
        events.hpp     <- интерфейс event system
        linemngr.hpp   <- менеджер вывода (очистка, отрисовка)
        variables.hpp  <- переменные проекта

main.cpp            <- точка входа, инициализация системы
build.bat           <- сборка Windows
build.sh            <- сборка Linux
```

## 🧩 Требования

- Windows
    "MSYS2 / MinGW-w64"
    "g++"
- Linux
    "g++"

## 🚀 Сборка

- Windows
    "build.bat"
- Linux
    "./build.sh"
